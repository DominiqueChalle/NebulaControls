# NebulaControls v1.0.0-beta

First distributable beta package for NebulaControls.

## Package

```text
NebulaControls.1.0.0-beta.nupkg
```

## Highlights

- Packable WPF control library targeting `net10.0-windows`
- Three themes:
  - `NebulaDarkPurple`
  - `NebulaDarkBlue`
  - `NebulaLightPurple`
- Runtime theme switching through `NebulaThemeManager`
- Public demo application
- Core controls for actions, inputs, selection, feedback, collections, data, and layout
- Control headers documenting name, version, and short description

## Included Control Areas

- Buttons, badges, alerts, dialogs, toasts, rating, spinner
- TextBox, PasswordBox, SearchBox, NumericUpDown
- DatePicker, TimePicker, DateTimePicker
- CheckBox, RadioButton, ToggleButton, ComboBox, Chip
- ListBox, TreeView, DataGrid
- TabControl, Expander, GroupBox, Avatar, ToolTip, Menu, ContextMenu

## Beta Notes

- This beta is intended for early testing and feedback.
- `NebulaDataGrid` is validated for local editing, add-row scenarios, and an SQLite-backed demo workflow.
- `NebulaTabControl` is currently top-only.
- Some controls may receive visual and behavioral refinements before a stable `1.0.0`.

## Installation

Add the package reference:

```xml
<PackageReference Include="NebulaControls" Version="1.0.0-beta" />
```

Load a theme and the global controls dictionary in `App.xaml`:

```xml
<ResourceDictionary Source="pack://application:,,,/NebulaControls;component/Themes/NebulaDarkPurple/Theme.xaml" />
<ResourceDictionary Source="pack://application:,,,/NebulaControls;component/Controls/NebulaControls.xaml" />
```
