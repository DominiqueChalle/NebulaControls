# NebulaControls v1.0.2-beta

Third public beta package for NebulaControls.

## Package

```text
NebulaControls.1.0.2-beta.nupkg
```

## Highlights

- Reworks `NebulaTimePicker` into a clock-based 24h picker with drag selection.
- Keeps `NebulaDatePicker` and `NebulaTimePicker` as the official picker controls.
- Removes the experimental `NebulaDateTimePicker` from the public control surface.
- Removes the internal `Picker Lab` view from the public demo.
- Keeps `DatePicker + TimePicker` as the recommended composition for date and time scenarios.

## Install

```xml
<PackageReference Include="NebulaControls" Version="1.0.2-beta" />
```

## Beta Notes

- This beta remains intended for early feedback.
- `NebulaTimePicker` is now visually closer to the planned picker direction, but may still receive visual refinements.
- `NebulaDateTimePicker` is intentionally not shipped in this beta to keep the official surface clean.
