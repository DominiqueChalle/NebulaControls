# NebulaControls v1.0.2-beta

Third public beta package for NebulaControls.

## Package

```text
NebulaControls.1.0.2-beta.nupkg
```

## Highlights

- Improves keyboard navigation and focus behavior across core controls.
- Adds clearer focus states for keyboard users.
- Adds typed `NebulaToastHost` APIs for info, success, warning and danger toasts.
- Refines demo focus and scroll behavior so keyboard navigation remains visible.
- Keeps the custom `NebulaDatePicker` and clock-based `NebulaTimePicker` as the official picker controls.
- Keeps `DatePicker + TimePicker` as the recommended composition for date and time scenarios.

## Install

```xml
<PackageReference Include="NebulaControls" Version="1.0.2-beta" />
```

## Beta Notes

- This beta remains intended for early feedback.
- Keyboard support is now much stronger, but beta feedback may still drive small refinements.
- `NebulaTimePicker` may still receive visual refinements in a later picker-focused pass.
- `NebulaDateTimePicker` remains intentionally out of the official surface for now.
