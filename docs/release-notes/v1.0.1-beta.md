# NebulaControls v1.0.1-beta

Second public beta package for NebulaControls.

## Package

```text
NebulaControls.1.0.1-beta.nupkg
```

## Highlights

- Adds `NebulaWindow` as a reusable themed window shell.
- Improves `NebulaToast` with stacked notifications, individual timers, close actions, and cleaner icons.
- Consolidates typography resources for more consistent text styling.
- Validates `NebulaDataGrid` with a realistic SQLite demo scenario.
- Refines `NebulaLightPurple` contrast for buttons, chips, toggle buttons, and menus.

## Install

```xml
<PackageReference Include="NebulaControls" Version="1.0.1-beta" />
```

## Beta Notes

- This beta remains intended for early feedback.
- Some controls may still receive visual and behavioral refinements before a stable release.
- `NebulaDataGrid` sorting, filtering, and row deletion are planned for later versions.
