# NebulaControls

NebulaControls is a WPF control library containing reusable Nebula-themed controls, styles, templates, and theme resources.

## Repository Layout

- `src/NebulaControls`: packable WPF control library
- `samples/NebulaControls.Demo`: public demo application using a project reference to the library
- `docs`: usage notes and extraction documentation
- `artifacts/packages`: local NuGet packages generated by `dotnet pack`

## Themes

The default theme is `NebulaDarkPurple`. `NebulaDarkBlue` and `NebulaLightPurple` are also available.

Themes can be switched at runtime:

```csharp
NebulaThemeManager.ApplyTheme(NebulaTheme.NebulaDarkPurple);
NebulaThemeManager.ApplyTheme(NebulaTheme.NebulaDarkBlue);
NebulaThemeManager.ApplyTheme(NebulaTheme.NebulaLightPurple);
```

## Build

```powershell
dotnet build NebulaControls.slnx
```

## Local Package

```powershell
dotnet pack src/NebulaControls/NebulaControls.csproj
```

The package is written to `artifacts/packages`.
